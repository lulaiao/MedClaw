# MedClaw

MedClaw is a public skills repository for medicine-focused AI workflows.

It is designed to collect reusable, structured, and maintainable skills for medical literature retrieval, clinical guideline reading, biomedical text organization, disease and drug terminology normalization, evidence synthesis, and other medicine-related tasks.

The goal of this repository is to provide a practical and extensible foundation for building medicine-oriented AI skill packages in a clear and reusable format.

---

## Overview

MedClaw focuses on medicine-specific tasks that benefit from structured prompting, explicit scope control, and cautious handling of medical information.

This repository is intended for skills related to areas such as:

- medical literature search and summarization
- clinical guideline retrieval and interpretation
- disease, symptom, and drug terminology normalization
- biomedical text extraction and organization
- clinical note structuring
- evidence-based medical writing support
- biomedical knowledge synthesis

MedClaw is not intended to replace clinical judgment, formal diagnosis, or individualized treatment decisions. Skills in this repository should support information organization and evidence retrieval rather than provide unsafe or overconfident medical conclusions.
